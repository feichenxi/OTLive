# ESP32-S3 OTA 一键更新工具

## 文件说明

- **ota_updater.exe** - 一键更新工具主程序
- **update.bin** - 待更新的固件文件（需自己准备）

## 使用方法

### 1. 准备固件文件

1. 将编译好的固件文件重命名为 `update.bin`
2. 将 `update.bin` 和 `ota_updater.exe` 放在同一目录

### 2. 运行更新工具

1. 双击运行 `ota_updater.exe`
2. 工具会自动：
   - 检测固件文件
   - 连接后端（默认：http://localhost:5000
   - 获取在线设备列表
   - 上传固件到后端固件库
   - 自动生成版本号（格式：vYYYYMMDD_序号）
   - 批量更新所有在线设备
   - 显示更新结果

## 版本号规则

自动生成的版本号格式：`vYYYYMMDD_序号

例如：
- 2026年4月19日第1个版本：v20260419_1
- 2026年4月19日第2个版本：v20260419_2
- 2026年4月20日第1个版本：v20260420_1

## 后端地址配置（可选）

如需修改后端地址，可以编辑代码中的 `BACKEND_URL` 变量，然后重新打包。

## 重新打包步骤：
1. 修改 `ota_updater.py` 中的 `BACKEND_URL`
2. 运行 `build_ota_updater.bat`

## 注意事项

1. **必须确保后端服务正在运行
2. 设备必须在线才能更新
3. 更新过程中请勿关闭工具
4. 更新成功后设备会自动重启
5. 固件文件必须命名为 `update.bin`
