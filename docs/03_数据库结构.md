# OTLive 数据库结构说明

## 一、数据库概述

OTLive系统使用两种数据库：
1. **SQLite (config.db)**：本地配置数据库，存储房间、设备、礼物等配置信息
2. **MySQL (live)**：远程消息数据库，存储互动消息和命令

---

## 二、SQLite数据库结构 (config.db)

### 2.1 rooms表（房间表）

存储直播间配置信息。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER PRIMARY KEY | 房间ID |
| name | TEXT NOT NULL | 房间名称 |
| ip | TEXT NOT NULL | 房间IP地址 |
| port | INTEGER NOT NULL DEFAULT 8080 | 房间端口 |
| online | BOOLEAN DEFAULT 0 | 在线状态 |
| rssi | INTEGER DEFAULT -100 | 信号强度 |
| sort_order | INTEGER NOT NULL | 排序顺序 |
| intercom_volume | INTEGER DEFAULT 50 | 对讲音量 |
| last_seen | REAL | 最后在线时间 |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 更新时间 |
| voice_status | BOOLEAN DEFAULT 0 | 语音状态 |
| voice_prompt | TEXT DEFAULT '' | 语音提示词 |
| voice_sample | TEXT DEFAULT '' | 语音样本 |
| voice_id | TEXT DEFAULT '' | 语音ID |
| voice_provider | TEXT DEFAULT 'fishaudio' | 语音提供商 |

**索引：**
- `idx_rooms_sort_order`：sort_order字段索引
- `idx_rooms_online`：online字段索引

### 2.2 devices表（设备表）

存储设备配置信息。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | 设备ID |
| room_id | INTEGER NOT NULL | 房间ID（外键） |
| name | TEXT NOT NULL | 设备名称 |
| label | TEXT NOT NULL | 设备标签 |
| pin | INTEGER NOT NULL | GPIO引脚号 |
| trigger_on_duration | INTEGER DEFAULT 3 | 触发开启时长（秒） |
| trigger_off_duration | INTEGER DEFAULT 1 | 触发关闭时长（秒） |
| enabled | BOOLEAN DEFAULT 0 | 是否启用 |
| gift_event | TEXT | 礼物事件 |
| trigger_remaining_count | INTEGER DEFAULT 0 | 剩余触发次数 |
| trigger_state | BOOLEAN DEFAULT 0 | 触发状态 |
| last_trigger_time | TIMESTAMP | 最后触发时间 |
| next_trigger_time | TIMESTAMP | 下次触发时间 |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- `idx_devices_room_id`：room_id字段索引

**外键：**
- `room_id` 关联 `rooms(id)` ON DELETE CASCADE

### 2.3 gifts表（礼物表）

存储礼物配置信息。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER PRIMARY KEY | 礼物ID |
| name | TEXT NOT NULL | 礼物名称 |
| value | INTEGER NOT NULL | 礼物价值 |
| level | TEXT NOT NULL | 礼物等级 |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 更新时间 |

### 2.4 gift_triggers表（礼物触发配置表）

存储礼物触发配置。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | 配置ID |
| room_id | INTEGER NOT NULL | 房间ID（外键） |
| gift_name | TEXT NOT NULL | 礼物名称 |
| trigger_count | INTEGER NOT NULL DEFAULT 1 | 触发次数 |
| device_type | TEXT DEFAULT 'main' | 设备类型 |
| device_config | TEXT | 设备配置（JSON） |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- `idx_gift_triggers_room_id`：room_id字段索引
- `idx_gift_triggers_gift_name`：gift_name字段索引
- `idx_gift_triggers_device_type`：device_type字段索引

**外键：**
- `room_id` 关联 `rooms(id)` ON DELETE CASCADE

**唯一约束：**
- `(room_id, gift_name)` 组合唯一

### 2.5 gift_trigger_logs表（礼物触发日志表）

记录礼物触发历史。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | 日志ID |
| room_id | INTEGER NOT NULL | 房间ID（外键） |
| device_name | TEXT NOT NULL | 设备名称 |
| gift_name | TEXT NOT NULL | 礼物名称 |
| gift_count | INTEGER DEFAULT 0 | 礼物数量 |
| trigger_count | INTEGER DEFAULT 0 | 触发次数 |
| original_count | INTEGER DEFAULT 0 | 原始触发次数 |
| remaining_count | INTEGER DEFAULT 0 | 剩余触发次数 |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 创建时间 |

**索引：**
- `idx_gift_trigger_logs_room_id`：room_id字段索引
- `idx_gift_trigger_logs_created_at`：created_at字段索引（降序）

**外键：**
- `room_id` 关联 `rooms(id)`

### 2.6 trigger_queue视图（触发队列视图）

显示待触发的设备队列。

| 字段名 | 说明 |
|--------|------|
| id | 设备ID |
| room_id | 房间ID |
| name | 设备名称 |
| label | 设备标签 |
| pin | GPIO引脚号 |
| trigger_on_duration | 触发开启时长 |
| trigger_off_duration | 触发关闭时长 |
| trigger_remaining_count | 剩余触发次数 |
| trigger_state | 触发状态 |
| last_trigger_time | 最后触发时间 |
| next_trigger_time | 下次触发时间 |
| room_name | 房间名称 |
| room_ip | 房间IP |
| room_port | 房间端口 |

**筛选条件：**
- `enabled = 1`
- `trigger_off_duration > 0`
- `trigger_remaining_count > 0`

**排序：**
- `next_trigger_time ASC`

### 2.7 system_config表（系统配置表）

| 字段名 | 类型 | 说明 |

## 三、MySQL数据库结构 (live)

### 3.1 msg表（互动消息表）

存储直播间互动消息。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT AUTO_INCREMENT PRIMARY KEY | 主键ID |
| license_id | INT(11) DEFAULT NULL | 授权ID |
| room | VARCHAR(100) DEFAULT NULL | 房间ID |
| type | VARCHAR(50) DEFAULT NULL | 消息类型：msg/gif/num |
| name | VARCHAR(100) DEFAULT NULL | 用户名 |
| content | VARCHAR(255) DEFAULT NULL | 消息内容 |
| giftname | VARCHAR(100) DEFAULT NULL | 礼物名称 |
| giftcount | INT(5) DEFAULT 0 | 礼物数量 |
| giftdiamond | INT(5) DEFAULT 0 | 礼物单个价值 |
| trigger | TINYINT(1) DEFAULT 0 | 触发标记 |
| status | TINYINT(4) DEFAULT 0 | -3已合并、-2已过期、-1已跳过、0待处理、1已生文、9已播报
| txt | VARCHAR(255) DEFAULT NULL | 感谢文本 |
| wav | VARCHAR(255) DEFAULT NULL | 感谢音频 |
| created_at | DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP | 创建时间 |

**索引：**
- `idx_license_id`：license_id字段索引
- `idx_room`：room字段索引
- `idx_type`：type字段索引
- `idx_status`：status字段索引
- `idx_created_at`：created_at字段索引
- `idx_room_created`：(room, created_at)组合索引
- `idx_license_room`：(license_id, room, status)组合索引

### 3.2 com表（命令表）

存储设备命令。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT AUTO_INCREMENT PRIMARY KEY | 主键ID |
| type | VARCHAR(50) DEFAULT NULL | 命令类型 |
| room | VARCHAR(100) DEFAULT NULL | 房间IP |
| license_id | INT(11) DEFAULT NULL | 授权ID |
| command | TEXT DEFAULT NULL | 命令内容 |
| status | TINYINT(4) DEFAULT 0 | 状态：-1失败/0等待/1成功 |
| at | DATETIME DEFAULT CURRENT_TIMESTAMP | 时间 |

**索引：**
- `idx_status`：status字段索引
- `idx_at`：at字段索引
- `idx_license_room`：(license_id, room, status)组合索引

### 3.3 license表（系统授权表）

存储系统授权信息。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT AUTO_INCREMENT PRIMARY KEY | 主键ID |
| machine_code | VARCHAR(64) NOT NULL UNIQUE | 机器码(MD5) |
| machine_name | VARCHAR(100) DEFAULT NULL | 机器名称(备注) |
| customer_name | VARCHAR(100) DEFAULT NULL | 客户名称 |
| expire_date | DATE NOT NULL | 授权到期日期 |
| license_type | ENUM('trial', 'monthly', 'yearly', 'permanent') DEFAULT 'monthly' | 授权类型 |
| max_rooms | INT DEFAULT 12 | 最大房间数限制 |
| features | JSON DEFAULT NULL | 功能权限配置(JSON) |
| status | TINYINT DEFAULT 1 | 状态：0=禁用/1=启用 |
| created_at | DATETIME DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
| last_verify_at | DATETIME DEFAULT NULL | 最后验证时间 |
| verify_count | INT DEFAULT 0 | 验证次数 |
| remark | TEXT DEFAULT NULL | 备注 |

**索引：**
- `idx_machine_code`：machine_code字段索引
- `idx_status`：status字段索引
- `idx_expire_date`：expire_date字段索引

## 四、触发器

SQLite数据库包含以下自动更新触发器：

### 4.1 update_rooms_timestamp

自动更新rooms表的updated_at字段。

```sql
CREATE TRIGGER update_rooms_timestamp 
AFTER UPDATE ON rooms
BEGIN
    UPDATE rooms SET updated_at = CURRENT_TIMESTAMP WHERE id = NEW.id;
END;
```

### 4.2 update_devices_timestamp

自动更新devices表的updated_at字段。

```sql
CREATE TRIGGER update_devices_timestamp 
AFTER UPDATE ON devices
BEGIN
    UPDATE devices SET updated_at = CURRENT_TIMESTAMP WHERE id = NEW.id;
END;
```

### 4.3 update_gifts_timestamp

自动更新gifts表的updated_at字段。

```sql
CREATE TRIGGER update_gifts_timestamp 
AFTER UPDATE ON gifts
BEGIN
    UPDATE gifts SET updated_at = CURRENT_TIMESTAMP WHERE id = NEW.id;
END;
```
