1 路由器网络配置

⚠️ 重要提示：系统要求所有设备必须在同一网段内才能正常通信！

1.1 修改路由器网段

  1. 登录路由器管理后台（通常是 192.168.1.1 或 192.168.0.1）
  2. 找到 LAN口设置 或 局域网设置
  3. 将路由器IP地址改为 192.168.1.1
  4. 子网掩码保持 255.255.255.0
  5. 保存设置并重启路由器
💡 配置示例：
路由器IP：192.168.1.1
DHCP范围：192.168.1.100 ~ 192.168.1.200
子网掩码：255.255.255.0

1.2 配置WiFi名称和密码（必须）

📶 房间盒子将自动连接以下WiFi

WiFi名称 (SSID)
OTLive
WiFi密码
OTLive.666
⚠️ 必须配置：请将路由器的2.4G WiFi名称改为 OTLive，密码设为 OTLive.666，房间盒子才能自动连接！

2 房间盒子IP配置

⚠️ 重要：房间盒子接入WiFi后不会自动获得指定IP，必须在路由器中为每个设备绑定静态IP！

2.1 设置静态IP绑定

  1. 登录路由器管理后台
  2. 找到 DHCP静态IP分配 或 静态IP绑定 功能
  3. 查看已连接设备列表，找到各房间盒子的MAC地址
  4. 为每个房间盒子绑定对应的固定IP（见下表）
  5. 保存设置并重启房间盒子

💡 线色说明：输入火线通常为 黑色 或 蓝色，输出火线为 红色。请以实际标注为准。

3.1 功率限制

3-220V

≤1500W

≤50W
细线功率限制

3.2 接线步骤

  1. 确认断电：接线前务必关闭总电源，确保安全
  2. 接入火线：将外部火线接入房间盒子的黑色/蓝色线（入）
  3. 接出火线：将房间盒子的红色线（出）接到设备的火线端
  4. 零线处理：设备的零线直接接入总零线（房间盒子不控制零线）
  5. 检查接线：确认所有接线牢固，无裸露铜线
  6. 通电测试：打开总电源，观察房间盒子指示灯状态
⚠️ 安全警告：
• 接线操作必须由专业电工完成
• 220V电压有触电危险，操作时请做好绝缘防护
• 接线完成后请仔细检查，确认无误后再通电
• 请勿超过功率限制，否则可能损坏设备或引发火灾

4 启动控制软件

  1. 确保电脑已连接到 192.168.1.x 网段的路由器
  2. 双击运行 IOT_Control_Center.exe
  3. 等待软件启动，自动检测房间盒子在线状态
  4. 在控制面板中查看各房间在线状态（绿色表示在线）
💡 常见问题：
• 如果房间显示离线，请检查房间盒子是否通电、WiFi是否正常连接
• 确认电脑和房间盒子在同一网段（192.168.1.x）
• 检查路由器中静态IP绑定是否正确配置
• 检查防火墙是否阻止了软件网络通信
