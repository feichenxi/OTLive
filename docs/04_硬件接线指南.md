## 三、杜邦线颜色汇总表

| 颜色 | 用途 | 接线数量 |
|------|------|----------|
| 白色 | PCM5102 BCLK | 1 |
| 紫色 | PCM5102 LRCLK | 1 |
| 灰色 | PCM5102 DIN | 1 |
| 红色 | 电源(3.3V) | 1 |
| 蓝色 | 地(GND) | 2 |
| 黄色 | GPIO 19/20 (主控制) | 2 |
| 绿色 | GPIO 21/47/48 (可编程1-3) | 3 |
| 橙色 | GPIO 45/35/36 (可编程4-6) | 3 |
| 棕色 | GPIO 37/38/39 (可编程7-9) | 3 |
| 黑色 | GPIO 40/41/42 (可编程10-12) | 3 |

> **说明**: 可编程设备按每3个一组共用一种颜色（绿、橙、棕、黑），共可控制12个可编程设备。

## 四、注意事项

### 4.1 电气安全
1. 确保所有设备共地
2. 使用合适的电源适配器
3. 避免短路和过载
4. 检查接线后再通电

### 4.2 热管理
1. ESP32-S3长时间工作可能发热，注意散热
2. PCM5102大功率输出时需要散热片
3. Windows PC确保良好通风

### 4.3 音频安全
1. 避免长时间高音量播放
2. 注意喇叭功率匹配
3. 测试时逐步增加音量
